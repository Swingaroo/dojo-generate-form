Gform

Gform helps you create forms for your dojo application. Gform is best suited for administrative frontends that require the user to enter lots of data and don't call for any special conversion rate optimization features or fancy visual effects.

Advantages

  • Ideally, the software development effort for a single form is independent of its complexity. Usually it comes down to converting an existing meta model like a java class to a schema.
  • Gform supports a clean look and feel for every data model. The developer can but does not need to modify it.

Supported schemas

Json schema (v. 0.3) and gform schema are supported formats. The former is a standard schema for json data and supported by several libraries. The latter is a specialized schema that provides extra information to take advantage of all gform features such as 'groups'.
