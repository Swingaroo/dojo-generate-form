Gform

Gform helps you creating forms for your dojo application. Gform is best-suited for administrative frontends that require the user to enter lots of data and the layout does not require any special conversion rate optimization features like fancy visual effects.

Advantages

  • The advantage of gform is that your development effort is indepedent from the complexity of your data model.
  • You don't need to worry about layouting. Gform supports a clean look and feel for any data model.

Use case

  • Gform makes it easy to create complex forms. It is best-suited for administrative front ends.
  • A gform form is made to read write json object via javascript. Posting the gform like an html form is currently not supported.

Schema

Json schema (v0.3) and gform schema are supported formats. The former is a standard schema for json data and supported by a lot of libraries. The latter is a specialized schema that provides extra information to take advantage of all gform features like 'groups'.
