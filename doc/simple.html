Gform

Gform helps you creating forms for your dojo application. The form is created according to your data model's schema. Json schema (v0.3) and gform schema are supported formats. The former is a standard schema for json data and can be created from server side data models. The latter is a specialized schema that provides extra information to take advantage of all gform features like 'groups'.
